# Artifact Availability Template

For the paper, a concise artifact sentence can use the following style:

> The source code, data, and/or other artifacts have been made available at `https://github.com/<your-org>/<your-repo>`.

Recommended repository checklist before you expose the link:

1. Keep a top-level `README.md` with environment, data, and reproduction commands.
2. Add a license file.
3. Add a citation file such as `CITATION.cff`.
4. Separate code, data instructions, and examples.
5. Avoid committing heavy outputs, checkpoints, and temporary caches.
